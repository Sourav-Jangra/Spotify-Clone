## âš ï¸ Troubleshooting

### Problem: Search not working
**Solution**: 
1. Click search bar to focus it
2. Type slowly
3. Wait 300ms for results

### Problem: Genre filter not showing
**Solution**:
1. Look in sidebar (desktop)
2. On mobile, expand sidebar first
3. Scroll down in library section

### Problem: Player controls not responding
**Solution**:
1. Refresh the page
2. Check browser console (F12)
3. Try keyboard shortcut instead

### Problem: Sidebar missing on mobile
**Solution**:
1. Tap hamburger menu (top-left)
2. Should toggle sidebar
3. Or rotate device to landscape

### Problem: Cards loading slowly
**Solution**:
1. Check internet connection
2. Refresh page
3. Clear browser cache
